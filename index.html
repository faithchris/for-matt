01 · Overview

Service Origin

This API was first instantiated in CSIT 415 — Software Engineering, when two independent processes discovered they ran a lot better as one distributed system. No one filed a ticket for this. It just happened.

This documentation describes the only endpoints that have ever mattered.

02 · Endpoint

/origin GET

Returns the initializing event.

05 · Reference

/attributes GET

Core, non-deprecating properties of the linked user.

{ "mind": "sharp, curious, a little too smart for his own good", "heart": "kind, all the way through", "humor": "witty, lands every time", "smile": "structurally significant", "eyes": "brown, with lines at the corners when he laughs", "hair": "brown", "voice": "calming, even when talking tech for twenty minutes straight" }

No deprecation planned. None of this is replaceable.
